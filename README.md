# CyberShield — Security Toolkit

A zero-dependency, single-file cybersecurity toolkit. Opens directly in any browser.

## Tools Included
- 🎣 **Phishing Analyser** — AI-powered email threat detection with risk scoring
- 🔐 **Password Vault** — AES-256-GCM encrypted credential storage
- 🌐 **IP/Domain Recon** — Geolocation and network intelligence
- 💀 **Breach Checker** — Check passwords against known data breaches
- **#️⃣ Hash Generator** — MD5, SHA-1, SHA-256, SHA-384, SHA-512

## Tech Stack
- Vanilla HTML / CSS / JavaScript
- Web Crypto API (AES-256-GCM encryption)
- Puter.js (free AI analysis)
- HaveIBeenPwned API
- ipinfo.io API

## Usage
Just double-click `cybershield.html` — no install, no server needed.
